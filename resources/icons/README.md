# Application Icons

This directory contains icon files for the ReqCockpit application.

## Icon Files

- `app_icon.png` - Main application icon (256x256)
- `new_project.png` - New project icon
- `open_project.png` - Open project icon
- `import.png` - Import icon
- `export.png` - Export icon
- `refresh.png` - Refresh icon
- `settings.png` - Settings icon

## Icon Format

All icons should be in PNG format with transparency support.

## Size Guidelines

- Toolbar icons: 24x24 or 32x32 pixels
- Menu icons: 16x16 pixels
- Application icon: 256x256 pixels
